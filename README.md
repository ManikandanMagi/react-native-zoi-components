# react-native-zoi-components

To create common components for mobile app

## Installation

```sh
npm install react-native-zoi-components
```

## Usage

```js
import { ZoiComponentsView } from "react-native-zoi-components";

// ...

<ZoiComponentsView color="tomato" />
```
